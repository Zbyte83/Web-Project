# Fundamentals

Esta sección contiene mis primeros proyectos web desarrollados con HTML y CSS.

El objetivo de estos proyectos es aprender los fundamentos del desarrollo web, comprender la estructura de una página y practicar la creación de interfaces sencillas.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## Contenidos trabajados

* Estructura básica de una página web.
* Etiquetas semánticas de HTML.
* Selectores y propiedades CSS.
* Posicionamiento y maquetación básica.
* Organización de archivos y recursos.

## Proyectos

Cada carpeta contiene un proyecto independiente realizado durante mi proceso de aprendizaje.

## Nota

Estos proyectos representan mis primeros pasos en el desarrollo web y sirven como referencia de mi evolución hacia proyectos más avanzados.
