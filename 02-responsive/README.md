# Responsive

Esta sección contiene proyectos web desarrollados con HTML, CSS y JavaScript, enfocados en la creación de interfaces adaptables a diferentes dispositivos y tamaños de pantalla.

El objetivo de estos proyectos es mejorar mis conocimientos de diseño responsive, organización del código y desarrollo de experiencias de usuario más completas.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## Contenidos trabajados

* Responsive Design.
* Flexbox y CSS Grid.
* Media Queries.
* Navegación adaptable a dispositivos móviles.
* Organización modular de archivos.
* Interactividad con JavaScript.

## Objetivos

* Crear interfaces adaptables para escritorio, tablet y móvil.
* Mejorar la experiencia de usuario en diferentes dispositivos.
* Aplicar buenas prácticas de maquetación y diseño.
* Desarrollar proyectos más completos y estructurados.

## Proyectos

Cada carpeta contiene un proyecto independiente con sus propios recursos y funcionalidades.

## Nota

Estos proyectos representan una etapa de evolución respecto a los fundamentos del desarrollo web, incorporando diseño responsive y una mayor interacción mediante JavaScript.
