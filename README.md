# Web Projects

Colección de proyectos frontend desarrollados con HTML, CSS y JavaScript.

Este repositorio contiene diferentes páginas web y mini proyectos creados para practicar, experimentar y mejorar habilidades de desarrollo frontend.

## Estructura del proyecto

```txt
web-projects/
├── fundamentals/
├── responsive/
├── components/
└── applications
```

## Tecnologías

- HTML5
- CSS3
- JavaScript
- WebComponents

## Objetivo

Crear y mejorar proyectos frontend de forma progresiva, escalando la web principal y añadiendo nuevas funcionalidades y componentes reutilizables.

## Proyectos

Los proyectos estarán disponibles desde la página principal del repositorio.
